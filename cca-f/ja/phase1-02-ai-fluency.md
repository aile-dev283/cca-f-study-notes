## カリキュラム

### レッスン 01: AI Fluency 入門

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291863>  

AI Fluency は、AI tools と効果的に協働するための実践的な能力である。単に button や prompt の書き方を知ることではなく、どの作業を AI に任せるか、どう説明するか、出力をどう評価するか、責任をどう持つかを判断できることを指す。

このコースでは、4D Framework（Delegation、Description、Discernment、Diligence）を通じて、AI と協働するための土台を学ぶ。

### レッスン 02: なぜ AI Fluency が必要なのか

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291870>  

AI は強力だが、万能ではない。AI をうまく使うには、task selection、context sharing、quality judgment、risk awareness が必要である。Fluency がないと、AI に不適切な仕事を任せたり、誤った出力を信じたり、責任の所在が曖昧になったりする。

AI Fluency は、効率だけでなく、品質、安全性、倫理性を高めるための能力である。

### レッスン 03: 4D Framework

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291876>  

4D Framework は、AI と協働する力を4つに分ける。

| D | 意味 | 内容 |
|---|------|------|
| Delegation | 委任 | 人間がやること、AI がやること、協働することを切り分ける |
| Description | 説明 | AI に task、context、criteria、format を伝える |
| Discernment | 見極め | AI の output、process、assumptions を批判的に評価する |
| Diligence | 誠実さ | AI 利用の責任、透明性、安全性、倫理性を守る |

4つは独立ではなく循環する。何を任せるかを決め、明確に説明し、出力を評価し、責任ある使い方に調整する。

### レッスン 04: Generative AI の基礎

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291879>  

Generative AI は、入力された context と training に基づいて、新しい text、code、images などを生成する。LLM は確率的に次の token を予測するが、単なる autocomplete ではなく、広い知識と pattern を使って推論・要約・変換・生成を行う。

ただし、AI は世界を直接理解しているわけではなく、誤りや作り話をすることがある。高 stakes の判断では検証が必要である。

### レッスン 05: capabilities と limitations

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291880>  

AI は要約、drafting、brainstorming、classification、translation、coding assistance、analysis などに強い。一方で、最新情報、正確な事実、専門的判断、倫理的判断、機密情報の扱いには注意が必要である。

能力と限界を理解すると、AI に任せる task の選び方が良くなる。

### レッスン 06: Delegation を詳しく見る

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291883>  

Delegation は、仕事を人間と AI の間でどう分担するかを決める力である。AI に任せるべき task は、目的が明確で、context を与えられ、output を評価できるものが多い。

人間が担うべきものには、最終判断、価値判断、責任ある意思決定、曖昧な stakeholder 調整などがある。

### レッスン 07: project planning と Delegation

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291886>  

project planning では、作業を分解し、AI が支援できる部分を見つける。research、outline、draft、risk list、review checklist、test idea などは AI に向く。一方で、最終的な trade-off 判断や承認は人間が担う。

良い delegation は、「丸投げ」ではなく、明確な outcome と review point を持つ共同作業である。

### レッスン 08: Description を詳しく見る

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291891>  

Description は、AI に何をしてほしいかを伝える力である。良い prompt は、role、goal、context、task、constraints、examples、output format を含む。

AI は context を推測できるが、推測に頼るほど output はぶれやすい。必要な背景を明示し、不要な情報を減らす。

### レッスン 09: 効果的な prompting techniques

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291895>  

効果的な prompting では、次を意識する。

- task を具体化する。
- audience と purpose を伝える。
- format を指定する。
- examples を与える。
- constraints と non-goals を明示する。
- 最初の output を見て反復する。

prompt は一度で完成させるものではなく、conversation を通じて改善するものと考える。

### レッスン 10: Discernment を詳しく見る

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291898>  

Discernment は、AI の output を批判的に評価する力である。正確か、十分か、偏りがないか、目的に合うか、根拠があるかを確認する。

評価の観点:

- factual accuracy
- completeness
- relevance
- tone and style
- hidden assumptions
- risk and safety

AI の自信ある言い方は正しさを保証しない。重要な点は source や independent check で確認する。

### レッスン 11: Description-Discernment loop

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291901>  

Description と Discernment は loop として働く。prompt を書き、output を見て、何が足りないかを評価し、prompt や instruction を改善する。

この loop により、AI との作業は one-shot ではなく iterative refinement になる。feedback は具体的にするほど良い。「短く」より「冒頭2段落を削り、結論を action-oriented にする」の方が効果的である。

### レッスン 12: Diligence を詳しく見る

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291904>  

Diligence は、AI を責任を持って使う力である。機密情報、個人情報、bias、copyright、透明性、accountability に注意する。

実践例:

- AI が関与した成果物であることを必要に応じて明示する。
- high-stakes な判断では人間の review を入れる。
- private data を最小限にする。
- 不確実性や limitations を共有する。
- AI output をそのまま自分の判断として扱わない。

### レッスン 13: まとめ

**URL:** <https://anthropic.skilljar.com/ai-fluency-framework-foundations/291908>  

AI Fluency は、AI を便利な tool として使うだけでなく、仕事の設計、品質判断、責任ある運用まで含む能力である。4D Framework は、AI と人間の協働を考えるための実践的な地図になる。
