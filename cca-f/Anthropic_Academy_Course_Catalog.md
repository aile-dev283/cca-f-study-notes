## Level 100 – Foundational

| Course | Description | Product / persona |
|--------|-------------|-------------------|
| AI Fluency: Framework & Foundations | Learn to collaborate with AI systems effectively, efficiently, ethically, and safely. | AI fluency / Professional, Personal |
| Claude 101 | Learn how to use Claude for everyday work tasks, understand core features, and explore resources for more advanced learning on other topics. | AI fluency / Professional |
| Claude with Amazon Bedrock (first 4 sections) | Introduction to working with Anthropic models through Amazon Bedrock. Covers setup, authentication, and foundational concepts including prompt engineering and evaluations. | Claude API and models on Bedrock / builders & devs on AWS |
| Claude with Google Cloud Vertex AI (first 4 sections) | Introduction to working with Anthropic models through Google Cloud's Vertex AI. Covers setup, authentication, and foundational concepts including prompt engineering and evaluations. | Claude API and models on Vertex AI / builders & devs on GCP |
| Building with the Claude API (first 4 sections) | Introduction to working with Anthropic models using the Claude API. Covers authentication, basic prompting, and core functionality including prompt engineering and evaluations. | Claude API / builders & devs |

## Level 200 – Intermediate

| Course | Description | Product / persona |
|--------|-------------|-------------------|
| Introduction to Model Context Protocol | Learn to build MCP servers and clients from scratch using Python. Master tools, resources, and prompts to connect Claude with external services. | MCP / Engineering |
| Claude with Amazon Bedrock (remaining sections) | Advanced topics for Amazon Bedrock integration including tool use, RAG, building agents, production patterns and optimization techniques. | Claude Developer Platform / Engineering |
| Claude with Google Cloud Vertex AI (remaining sections) | Advanced topics for Vertex AI integration including tool use, RAG, building agents, production patterns and optimization techniques. | Claude Developer Platform / Engineering |
| Building with the Claude API (remaining sections) | Advanced API topics including tool use, RAG, building agents, production patterns and optimization techniques. | Claude Developer Platform / Engineering |
| Claude Code in Action | Practical guide to integrating Claude Code into development workflows covering context management, hooks, custom commands and the Agent SDK. | Claude Code / Engineering |
